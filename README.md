# toj-vitamin

Маркетплейс витаминов и БАДов GLS. 

### Основные функции:
- Улучшенный интерфейс в стиле Apple (Glassmorphism).
- Интеллектуальный квиз-диагностика для подбора витаминов.
- Полноценная корзина и оформление заказа через WhatsApp.
- Оптимизированный каталог из 107 товаров с актуальными ценами (TJS).
- Полная локализация (Русский / Таджикский).
